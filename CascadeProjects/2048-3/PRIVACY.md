# Politique de Confidentialité - Poz

**Dernière mise à jour : Janvier 2026**

## Introduction

Poz ("nous", "notre" ou "l'application") s'engage à protéger votre vie privée. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations personnelles.

## Informations Collectées

### Informations que vous nous fournissez
- **Compte utilisateur** : Email, nom d'utilisateur
- **Données de profil** : Informations physiques (poids, taille, objectifs)
- **Données d'entraînement** : Exercices, séries, répétitions, poids utilisés
- **Contacts** : Liste de contacts (avec votre permission) pour vous connecter avec vos amis

### Informations collectées automatiquement
- **Données d'utilisation** : Interactions avec l'application
- **Données techniques** : Type d'appareil, version iOS, identifiants uniques

## Utilisation des Données

Nous utilisons vos informations pour :
- Fournir et améliorer nos services
- Personnaliser votre expérience d'entraînement
- Analyser votre progression sportive
- Vous connecter avec vos amis utilisant l'application
- Assurer la sécurité et prévenir la fraude

## Partage des Données

Nous ne vendons jamais vos données personnelles. Vos informations peuvent être partagées uniquement dans les cas suivants :
- **Avec votre consentement** : Lorsque vous choisissez de partager vos données avec vos amis
- **Fournisseurs de services** : Firebase/Google pour l'hébergement et l'authentification
- **Obligations légales** : Si requis par la loi

## Sécurité des Données

Nous utilisons des mesures de sécurité standards de l'industrie pour protéger vos données :
- Chiffrement des données en transit (SSL/TLS)
- Authentification sécurisée via Firebase
- Stockage sécurisé dans le cloud Google

## Vos Droits

Vous avez le droit de :
- **Accéder** à vos données personnelles
- **Modifier** vos informations de profil
- **Supprimer** votre compte et toutes vos données
- **Exporter** vos données d'entraînement
- **Retirer votre consentement** à tout moment

## Conservation des Données

Nous conservons vos données tant que votre compte est actif. Si vous supprimez votre compte, vos données seront supprimées dans un délai de 30 jours.

## Données des Mineurs

Poz est destiné aux utilisateurs de 13 ans et plus. Nous ne collectons pas sciemment d'informations auprès d'enfants de moins de 13 ans.

## Modifications

Nous pouvons mettre à jour cette politique de confidentialité. Les modifications importantes vous seront notifiées via l'application.

## Contact

Pour toute question concernant cette politique de confidentialité :
- **Email** : privacy@poz-app.com
- **Support** : support@poz-app.com

## Conformité RGPD

Pour les utilisateurs européens, nous respectons le Règlement Général sur la Protection des Données (RGPD). Vous disposez de droits supplémentaires concernant vos données personnelles.
